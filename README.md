# Example request

Install pipenv
```aidl
pip3 install pipenv
```

Install modules
```aidl
pipenv install
```

Activate virtual env
```aidl
pipenv shell
```

To turn off the env type
```aidl
exit
```

To run the django
```aidl
python manage.py runserver
```
